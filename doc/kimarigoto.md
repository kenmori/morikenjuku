# もりけん塾の決まりごと

お互い心地よく学習する、教えるための決まりごとです

## 前提

以下のことは大前提のスタンスで、教える側、教わる側どちらにも言えることです

- 健康を害さない程度に頑張りましょう。健康第一
- 心的悪影響があった場合、全てを投げ出して逃げていい、拒否していい
- こちらに書かれている全てのことは話し合い可能です
- なかなか時間が取れない、⽅向性が違ってきた場合は⼀旦離れる。⼟⽇しかできない、学習の時間が確保できない、仕事で忙しい⽅も同様。ここにいることが⽬的ではなく、学習習慣を築き上げることこそエンジニアとしてスキルを上げる最低限のやることだと考えるからです
- こちらに書いてあることは4ヶ月に1度大きな変更があります。小さい変更は都度あります

## 定義

- `JS課題`・・・こちらが提供するJavaScriptの課題。基本全ての方に取り組んでもらっています。その後の課題に関しては個々の貢献度や条件、その時の塾の方針で変わります。
- `言語チャレンジ`・・・こちらが提供するJSメソッドを使いinput(引数)を元にoutput(返値)を出力(console.logで出力)してもらう問題

- `もりけん塾朝活部`・・・早朝から10時までもくもくする場合。サテライトサーバーの`もりけん塾朝活部`のボイスチャンネルでもくもくしてもらっています。

- `codewars部`・・・[codewars](https://www.codewars.com/dashboard)にアカウントを作ってもらい(githubアカウントで)その後チャレンジしていくものです

- `もくもく塾生`・・・サテライトサーバーのもくもく塾生部屋だけ借りる塾生。学習の習慣を作ってもらうことを目的としている ([mokumokujukusei.md](https://github.com/kenmori/morikenjuku/blob/main/doc/mokumokujukusei.md)参照)。

- `ジュニア塾生(以下塾生)`・・・JS課題をしている塾生。下記条件を満たす塾生。Twitterでの「塾生」募集はこれになります

- `シニア塾生(シニア)`・・・JS課題をある程度終えてジュニア塾生として終了、塾を離れずに個人で学習を続けていく方

## 学習ツール

- Mac(Winの方はご自身で置き換えてください)
- [Discord](https://discord.com/) 塾内のやりとりの際に必要です

- [codewars](https://www.codewars.com/dashboard) 任意の課外活動です。

## 塾生の方が守ること

Twitter内のDMでもお伝えした通り、以下のことを基本的に守っていただいています

- Twitterアイコンはご自身のお顔。(イラスト不可) 塾内のSNSはお顔がわかるものと本名のひらがな表示になります
- こちらが提供するJSの課題をやっていただくこと
- Discord内でアクティブになった場合(もくもくや朝活する)はご自身のtimesにやることやったことなど投稿してください。この期間が空くと声かけさせてもらいます
- 以下のアクションがあった場合Twitter上呟いてもらい `#もりけん塾`と`@terrace_tech`と`@教えてもらった方付き`をつけることにしています
  - 積み上げ
  - わからないことで質問し、教えてもらった、理解が進んだ時
  - 1on1後
  - 誰かにレビューしてもらった時
  - 勉強会に出席後1日以内に感想(任意)

- ブログ記事を書いたら記事下に[ブログへのリンク](http://kenjimorita.jp/)と[Twitterへのリンク](https://twitter.com/terrace_tech)をつけること

## 休学

思うように時間が取れない⽅はこれを選べます。⼀時的に学習を積み上げられない場合、やむを得ない場合、習慣が⾝に付いていなく、もくもく塾⽣にもなれそうにない⽅。
こちらは復帰することを期待しているものですのでその際復帰時期を決めてもらい、話し合い、合意された場合、休学になります。
## さよなら判断

前提で、「どちら」も「いつでも」さよならできます

- 下記禁止事項をした場合
- `「積み上げ」` `「アクティブ数」` `「貢献度」` `「意欲」`など総合的に判断して、こちらがコントロールできない、コーチングすることが億劫になった場合
- Twitter上のプロフにある`#もりけん塾`タグを外した場合

### 月一目標共有とは

もりたが月初に`#月一目標共有部屋`内で塾生に対してメンションで呼びかけますので反応、目標を投稿していただくことです。

### 積み上げとは

学習した際にTwitter上でタグ付けとメンションをして呟くことです

背景は、
もりけん塾では無料でやっている手前、その対価としてこの活動を広めていただける方を入塾してもらっています

上記のことをすることでお互いメリットがあると考えています
もし転職の際などでタグの印象を気にするようであれば相談の上判断しますのでおっしゃってください

### 塾に関する積み上げ以外は不要です

- 「積み上げ休みました」
- 「WP勉強しました」
- 「仕事で〜やってました」
- 仕事や別学習(※上記参照)を積み上げ内容と⼀緒にすること

### ご褒美講義

普段からのアクティビティとオーナーシップがある⽅はある⽇特別講義のお誘いをDMします。いつ誘いが来るかはわかりません

- レビューを積極的にしている
- 何かの頼み事に率先している
- Twitter上でこちらが恩恵を受けることを能動的にしてくれている
など感謝したい事が溜まった時にシークレットで開催します

### アクティブ数とは

Discord内がアクティブになった回数です。

### 貢献度とは

- 誰かのレビュー
- オーナーシップ(率先して塾内のことを決めていくこと。ドキュメントにPRを出すこと)
- 勉強会での発表
等
能動的にしていただいた回数です

これらのことは細かく数えていないですが、何か塾生がやりたいときやお願いしたい時、これらを判断材料にさせていただいています。また、今後の選抜されるときの判断になります。

## 禁止

一発退場事項です

- Twitterのアカウントを変えてしまうor削除、鍵付きにすること
- 無料で教えている手前、有料スクール、有料・無料サロンに入ること
- 他の塾生への勧誘や心理的安全性を乱す行為をすること
- ここで得た他企業とのやりとり情報やリソースは外へ漏すこと
- ⼈材への紹介や提案の禁止です。学習をする場として提供していて集まってもらっている為。具体的に外部の⼈と繋がることや取りまとめすることを禁⽌します。
また弊社が今春、⼈材紹介業務をするので競合となりえるので、宣伝や具体的な競合会社名を挙げることは控えてもらいます

## Q&A

- slack時代からいた塾生は毎日積み上げツイートなど条件になかったと思います。どうすればいいですか -> 2021年10月から一律変更になります。そしてその条件変更は常に応相談可能です。

- 塾生全員同じ決まりごとですか？ -> いえ違います。それぞれの塾生は条件が違いますが、それらの条件はこの **決まりごとの上に成り立ちます**。こちらは共通の決まりごとになります

- 正直土日しかアクティブになれてないのが申し訳ないです。どうしたらいいですか。 -> しょうがない。ただ土or日にアクティブになればいいです。土日しかアクティブにならないことがわかっている場合は、土日アクティブになった時に`ご自身のtimes`に投稿してください。一方`積み上げ報告`に関しては業務の事でもしてもらいたいです。
なぜならこちらの課題ができないことと、積み上げ報告ができないことは別なので気持ち悪さはあるがそうしてもらっています

- JS課題が終わったら何をすればいいのかわかりません -> 今今こちらが提供できる課題がないですが、気長に待っていただくかこんなことやりたいなどあれば条件に応じて作ります

- JS課題が終わると塾生でいるのかシニアになるのか選べるのでしょうか？ -> はい。

- JS課題が終わっていない場合はシニアになれるのでしょうか -> 課題の進み具合とこれまでの貢献度によります

- シニアは勉強会がゲスト参加になるとありましたが、勉強会後の課題のレビューや質問なども不可になりそうでしょうか？-> はい。ゲスト参加ですので終了後はできないです

- 1ヶ月向き合っているのですが課題が進みません。辞めた方がいいでしょうか
学習意欲に関して、今まで⾒てきて、1ヶ⽉に課題1つクリアぐらいだとフロントエンドになるには厳しいなと感じています。そのような⽅は何かが⾜りてないです
- 質問の量
- 時間の捻出
- やる気
ご⾃⾝が当てはまる場合相談してください。DMでもいいです。ギブアンドテイクですが何か考えたりします。(例えばAさんのように、課題はやらず今直⾯している問題を積み上げて代わりに私が技術を受ける等)
向き合って1ヶ⽉で1つできなかった⽅と、向き合ってなくてできなかった⽅は区別しています。前者の場合は問題ないです
相談してくれれば⼤抵のことを受け⼊れているつもりですので是非相談してください

- 他コミュニティへの参加したいです。-> 有料無料問わずこちらにご連絡ください。基本的には、無料でやっているもりけん塾が他コミュニティに入っているそこ方の面倒を見ることはしたくないし、シニアであれ、塾生であれ、自分はそっとさよならすると思います。無料でやっているコミュニティには寛容ですが、それも条件によります。ジャンルが違うところだったら応相談

- やめたい場合どうすればいいですか? -> もりけん塾の助けがいらなくなった場合もDM、もしくはプロフのタグをそっと外してください。違う言語をやりたい、試験がある、案件取れて忙しくなってきた、疲れた、など方向性が変わってきたらDMでもいいので教えてください。もしくは静かにフェードアウトしてもらっていいです

- サーバーサイドに関しての質問も可能ですか -> いえ。自分自身が専門的ではないので控えようと思います

- 何か課題とは別に頑張りたいことが出てきました。やめなければいけないですか？
 -> DMで相談してください。もしお互いの条件が合えば、いろいろな環境を用意できます。例えばいつも貢献度が高い方には、別途テキスト部屋を設けてやめなくてもいい環境を作ったりしています

- codewarsはアクティブに数えられますか？ -> いえ。これは任意でやってもいい課外活動の位置づけですので対象はあくまで課題が終わっていなければ課題、言語チャレンジや貢献活動です。
